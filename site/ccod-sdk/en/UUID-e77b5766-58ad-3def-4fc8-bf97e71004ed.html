Installing an SDK plug-in

In SDK version 3.0.0 CashComplete™ On-Device scans the \CashComplete\data\plugins folder for implementations of the IOJ plug-in. The plug-ins are automatically loaded unless blocked by the configuration in the com.scancoin.sdk.engine file.

You can place the plug-in jar file anywhere you want on the computer running CashComplete™ On-Device, but it is recommend that you save it to the \CashComplete\data\plugins folder.

Plug-ins saved to the data subfolder are automatically upgraded when you upgrade CashComplete™ On-Device.

Install an SDK plug-in

  1. Create a folder folder for the plug-in jar file. For example, C:\CashComplete\data\plugins, assuming that C:\CashComplete is the CashComplete™ On-Device installation folder.

  2. Save the plug-in jar file to the folder you just created.

  3. If you have no previous plug-in configuration, save the plug-in in the \CashComplete\data\plugins scanned by CashComplete™ On-Device for plug-ins to load.

  4. Optionally (earlier method), do the following:

    • Open the \CashComplete\resources\configurable\configuration\com.scancoin.sdk.engine in a text editor.

    • Add the plug-in to the file by specifying the jar file either as an absolute or relative path to the CashComplete™ On-Device installation folder. You must type the jar file path followed by the name of the main class, separated by a semicolon. The name should be fully qualified with the package name: PATHTOJAR;CLASSNAME.

    • Edit the configuration file to say: plugins = data\plugins\MyPlugin.jar;ThePluginClass

    • To load two plug-ins, edit the file to say something like: plugins = data\plugins\MyPlugin.jar;ThePluginClass,data\plugins\MyPlugin2.jar;ThePluginClass2.

  5. Restart the CashComplete™ On-Device software.
